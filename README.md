# BLITZ — Dodge or Die

A fast-paced arcade-style dodging game with a cyberpunk aesthetic. Navigate waves of enemies, build combos, and survive as long as you can.

## 🎮 Play Now

Open `blitz.html` in your web browser to start playing.

## Features

- **Progressive Difficulty**: Waves increase in intensity with more enemies and speed
- **Multiple Enemy Types**:
  - **Normal Enemies** (Red): Linear attackers that move toward you from the edges
  - **Homing Enemies** (Orange): Intelligent enemies that track your position (Wave 3+)
- **Combo System**: Chain enemy defeats for bonus points
- **Lives System**: 3 lives per game with visual heart indicators
- **Proper Wave Progression**: Game correctly starts at Wave 1 and progresses only when enemies are defeated
- **Retro Aesthetics**: Grid background, scanlines, glitch effects, and neon colors
- **Mobile Responsive**: Fully centered UI and controls optimized for mobile web browsers
- **Responsive Controls**: Keyboard (WASD/Arrow Keys) or touch/mouse drag support

## 🎯 How to Play

1. Click **PLAY** to start a new game
2. Avoid all enemies while staying on screen
3. Enemies spawn in waves of increasing difficulty
4. Defeat enemies by touching them (they disappear on contact)
5. Build combos for bonus points
6. Lose a life if hit 3 times - game over when lives reach 0

## ⌨️ Controls

- **WASD** or **Arrow Keys**: Move around the screen
- **Mouse/Touch Drag**: Move by dragging your pointer toward the direction you want to move

## 📊 Game Mechanics

- **Score**: Points earned for surviving and defeating enemies
- **Wave System**: Enemies spawn in progressively harder waves
- **Combo Multiplier**: Chain consecutive enemy defeats for bonus points
- **Invincibility Frames**: Brief invincibility after taking damage
- **Best Score**: Your high score is tracked and displayed on the game over screen

## 🎨 Visual Design

Built with:
- HTML5 Canvas for graphics
- CSS Grid background and scanlines for retro aesthetic
- Google Fonts (Orbitron, Share Tech Mono)
- Neon color scheme:
  - **Cyan** (#00f5ff): Player character
  - **Red** (#ff2244): Normal enemies
  - **Orange** (#ff8800): Homing enemies
  - **Yellow** (#ffdd00): Score and wave display
